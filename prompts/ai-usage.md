# AI Usage Notes

## Where AI Helped

- Planning the project structure and REST endpoints
- Designing the UI layout and styling ideas
- Drafting starter text for documentation

## Good Prompts

- "Design a REST API for a mobile webshop with sessions and roles."
- "List data entities for a phone store and show their relationships."
- "Create a bold, warm UI theme for a product catalog."

## Weak Prompts

- "Make it better." (too vague)
- "Do everything for me." (no constraints)
- "Fix it." (missing error context)

## Notes

- AI output was reviewed and adjusted to match the project requirements.
- All generated ideas were integrated manually into the repo.
